# 暗影指数

## 豪华疾行者外壳

外壳型号：刺客
外壳风格：暗影指数
外壳系列：沃伊德
<span style="color: rgb(26, 122, 215);">一一一一一一一一一一一一一一一一一</span>

暗影指数是一个精英疾行者组织。创立者身份不明,成员资格仅限授予一小部分经过严格筛选的疾行者,他们必须证明自己不仅能够在敌对区域生存,更能成为其中出类拔萃的一员。

除开声誉,暗影指数并没有实质性的影响力。他们不涉足阵营战争,也不会以高价出售成员资格。他们只专注于观察在敌对区域中挑战外壳极限的疾行者,发现满意人选后,就会通过秘密渠道向其发送暗影指数外壳方案,作为加入组织的欢迎礼。已知仅有极少数疾行者拒绝了暗影指数的邀请。
