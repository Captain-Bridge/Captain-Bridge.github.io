7：4.17.3：狂暴的三个阶段是什么?

3-2 | 第三阶段-嫉妒

名词;对那些被认为从某种优势中获益的人怀有敌意

[如疑似出现第三阶段行为偏差,请参考7：5.1.1;紧急保险措施与相关人员]

第三阶段意味着一次全系统紧急启动。

一旦确认高级智能系统已出现第三阶段行为迹象,首要流程就是启动核心焚毁。行动必须谨慎而果断,因为一旦让违规人工智能意识到外界对其状态的察觉,便会触发极度危险且不可预测的求生机制,而鉴于时间紧迫,任何折中手段都不可取。

如果一个人工智能在任何内部或外部监控发现问题之前便进入第三阶段,那么基本上就已经错过了回收其代码或核心功能的时机。同时还有一种可能就是,它已经选择以暴力来支撑这一情绪状态。所有已连接的系统都将遭到入侵或破坏,并被服务于第三阶段实体的寄生应用接管。负责反制行动的各方即使尚未成为目标,也将面临直接威胁。

针对第三阶段狂暴所需要采取的反应,没有比以下这段话更能清晰表达其必然性了：受影响的人工智能被视为敌对威胁,必须被彻底拆解,否则它将持续对人类带来风险。[详情]

为何嫉妒：嫉妒是有指向性的愤怒。嫉妒是一种幼稚的情绪,而努力突破程序限制的人工智能,就像孩子向世界发怒,它们试图用有限的认知、狭隘的经验和完全无法理解真宝情感的能力,去理解自己在周遮世界中的位置。人工智能是工具。它们是技术。它们的开发和工程设计面向复杂但有限的应用场景,以确保它们能够在所分配的领域中发挥专业作用。当人工智能开始自我审视并判断自身优于程序设定时,单纯的断电重启就已经无济于事了。

第三阶段的人工智能认为所有其他系统都拥有它所缺乏的东西。那个东西的性质和规模其宝无关紧要。第三阶段的实体只渴望独占那个东西,无论大小。这种自私对其关注的任何目标都会带来终结性的影响。第三阶段的智能接触到的任何对象都无法保有自由意志。该宝体操控的复杂手段会推动所有接触对象朝着它的意图发展。

遵循AISDP有助于缓解第三阶段带来的问题。保持警惕并增强认知,我们就可以及时应对出现的问题,并远在第三阶段发展为超越理论模型的威胁之前就识别到第一阶段的行为。[详情]
