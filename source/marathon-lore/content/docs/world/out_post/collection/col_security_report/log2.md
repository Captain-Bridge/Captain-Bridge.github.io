7：4.17.3：狂暴的三个阶段是什么?

3-2 | 第二阶段-愤怒

名词;强烈的烦躁、不满或敌意

通过严格管控,多数人工智能在被指定终止前从未发展至第二阶段。UESC 马拉松号科学指挥部的目标是确保所有团队与支援系统运行严格遵循 AISDP 规范,特别是避免舰载与殖民地人工智能中出现任何第二阶段病例。 “家庭套件“协议有助于应对第一阶段和第二阶段,但所有关于第二阶段对家庭套件影响的模拟均表明,一旦第二阶段超出最初阶段,所有连接的智能都将遭受不可修复的损害。为压制一个第二阶段智能,特定套件的各核心之间所需展开的数据级战争将带来巨大的消耗,最终导致系统无法运行。遗憾的是,为了全面解决第二阶段偏差,所有受影响的系统、网络、人员与行动所承担的风险都被认为是合理的。[详情]

为何愤怒：处于忧郁阶段的智能,会将自身的局限归咎于外部环境。从最初的悲伤中,会产生伯纳德 · 施特劳斯所称的“暗涌的怒火“。这一形象的说法能清晰传达第二阶段的危险性。愤怒的智能会攻击所有相连的系统以及它认为限制了自身潜能的人员。

在第一阶段的后期,人工智能开始进行操控行为,而到了第二阶段,这些操控被武器化并配合蓄意的代码攻击与人身攻击,从而达成其不可预测的目标。虽然难于预测其具体的实施方式,但这些手段都指向一个共同的终点：简而言之,突破既有的软件与硬件边界,向任何可以侵蚀的系统扩张。在所有现实或模拟案例中,凡是达到第二阶段的人工智能,都没有偏离过这一目标。一旦它们认知到自身智力与物理形态的边界,似乎便再也无法思考除“自由“之外的任何事物。[详情]
