行星迁移紧乱(PRD) ：了解和寻找支持

什么是行星迁移紧乱?

行星迁移紧乱(PRD)是一种部分人移居到新的行星后会患上的心理健康问题。PRD 并非单一疾病,而是一种症候群。有时人们也将 PRD 称为“舰震症“或“哭泣病“。

在完成跨行星移民后,遭遇到神经生物和心理方面的问题是很正常的。具体症状因人而异,可能包括抑郁、行为混乱、思维或语言混乱、人格解离、肾上腺素反应失调、非志愿反复记忆,精神失常、自残和睡眠瘫疲等等。大部分人都会能在无需介入的情况下,经过数周时间的调整期而自行康复。

如果在抵达六个月之后,您依然能体验到上述症状,那么您可能患上了 PRD。但事情并非没有转机：您还可以接受 PRD 治疗,而且这种疗法有着显著疗效。

您并不是孤立无援

大约每六名跨行星移民者中会有一名出现部分 PRD 症状。

有一些因素可能会导致更容易悦上 PRD,例如：标种类型的创伤、反复低温休眠、滥用药物、基因性易染病体质、曾接触野生植物以及未经授权的自我强化等等。

治疗会采用什么方式?

您的人工智能顾问或医疗专家会询问您的症状,以制订您的治疗方案。取决于您的症状的严重程度,他们可能会建议进行不同的治疗方法,如眼动脱敏和再建疗法、药物治疗、保守隔离或治疗性神经精神转移。

在经过治疗后,大部分人都报告说生活质量有所提升。

我还能成为我自己吗?

在经历 PRD 时,对自己的身份认同产生疑虑也是很正常的情况。大部分人在治疗期间或之后都能够与这种陌生感实现调和。

记住：即便你身处的环境发生了变化,但你依然是你。你的医疗团队会帮助你调适心理,接收你的新生活。
