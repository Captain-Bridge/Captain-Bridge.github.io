安全协议激活通知

NCCC 的各位成员,

我是里德将军。
我现在将针对先前的会议做出直接回应。我在此通知各位：在一小时内,我作为新卡斯卡迪亚及UESC 马拉松号安全指挥官安全指挥部负责人,将临时接管所有行星与舰船行动的全面临时全面指挥权。

根据指挥部行动协议要求,现对本组成员下达通知,以重申既定流程并确保人员服从性,从而顺利过渡至各自的危机管理职责中。

为安抚各方顾虑,我预计不会对公开尿面的指挥架构作出太多调整,如有变动也将控制在最小范围内。相关计划仍待最终敲定,但我们正根据局势加速推进、我也需要各位做到同样的效率一一及时执行接下来发布的各项指令,有助于公众在面对并理解此次损害规模时,继续维持对我们领导层的信心。

各位将在后续筒报中获悉详细内容一一在处理0307事件善后工作的同时,我们将尽一切努力维持正常的应急运作。无论是保障应急服务的顺利开展,还是维护殖民地的长期存续能力,都要求尽可能减少对公众日常生活的影响。

我知道这听上去似乎与现实状况背道而驰,但我向各位保证,承认这场悲剧的规模与影响,同时淡化具体细节并按标准程序应对,将使公众更容易接受我们的说法,而军安指挥部的公开接管反而会破坏这一效果。

接下来我们所做的一切都不会轻松,但它们势在必行。

第一阶段行动是立即管控所有受到此次事件直接影响的人员并安排问询。其中包括轨道目击者、港口区域入侵事件幸存者,以及所有可定位的异常对象。我们需要确认每个人具体目睹或经历了什么,以及他们可能接触过哪些对象。

此外,我已批准安全指挥部行动人员进驻所有打捞回收单位,以管控残骸的收集与保管。现阶段无法确认现场有哪些残留,因此必须尽一切努力确保100%回收所有非本地物质,包括生物性与机械性材料。

在本次接触期间,通讯能力遭到严重限制,而我们又已成功将行星情报套件系统完全分割,因此公众得到的信息混淆不清,我们可通过有策略地投放信息来掌控外界认知。

我们在这里打造的生活建立在信任与诚实的基础之上,所以我愿坦率承认一一是的,欺瞒令人遗憾,却是不可避免的手段。如果本组成员中有人对我们选定的行动方针产生动摇,请提醒自己,要守护的究竟什么。

在现阶段,马拉松号作为一舰船对我们而言已经失去作用,我们尚无法确定它是否还有恢复运作的一天。即便能够修复其家庭套件的损毁以及系统与驱动所遮受的破坏,也可能需要耗费数十年的时间。若将这次接触的真相以及被困此地、既无援助也无逃离手段的现实一并公之于众,那么别说维护和平,就连让公众维持日常生活重心都会变得近乎不可能。

我们不能冒险让这片殖民地因为恐惧而走向内部分裂。所以,我们必须控制信息传播。我们必须哀悼。必须恢复。我们必须放下过去、继续前进。同时,期盼曾发现我们的那个存在,不会再次找到我们。

相关命令、启用安排及信息管控细节将陆续传达。

如有任何问题或顾虑,请通过加密渠道直接与我办公室取得联系。

达维克 · B · 里德将军
新卡斯卡迪亚安全指挥部
安全协议激活通知
