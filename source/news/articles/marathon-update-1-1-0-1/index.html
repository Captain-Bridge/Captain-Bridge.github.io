Marathon Update 1.1.0.1

The one about Sentinel Mastery Entries

Marathon Dev Team June 9, 2026
updates616 words3 min read

Item Economy

  • Slightly reduced the chance for Prestige Shield to appear in all sources.
  • Slightly reduced the chance for Prestige Weapon Mods to appear in all sources.
  • Fixed various issues where Schemas granted more or fewer Credits than their listed value upon exfil.

Combat

Weapons

  • Fixed an issue where dragging a weapon mod from one equipped weapon to another could incorrectly show the mod as incompatible, even when it was valid for both weapons.
  • Fixed an issue with the D54 Battle Pistol Prestige mod perk “Daredevil Stock” where spam clicking could extend the super burst for the full duration of the magazine.

Chips

  • Fixed an issue where the Aftershock perk was expiring too quickly after taking explosive damage.

User Interface and Experience

  • Fixed an issue where Push-to-Talk would fail while in the prepare screen.
  • Updated item tooltip to not include certain actions while the multi-select is active in lobby.
  • Added missing Zone Status Effect “it’s Dark” on the Night Marsh preview.
  • Fixed an issue with the Cryo Maintenance (Cryo Archive 01) key description incorrectly listing the unlock steps.
  • Fixed an issue that caused ghosting/artificating in the Lobby while using AMD FSR with the resolution scaling set to quality or lower.
  • Fixed an issue where turning on Streamer Mode mid-run would display inconsistent names on the HUD.
  • Fixed an issue where the Season 2 Premium Rewards Pass purchase screen would display a LUX icon instead of a Rewards Pass Voucher icon.
  • Fixed an issue where Rewards Pass items couldn’t be selected when navigating from the “Vox Nocturna” Knife Style to Rewards Pass using the source link.

Factions

Sekiguchi Genetics

  • Fixed an issue where Capstone 6 could not be claimed after unlocking all the required upgrades.

Contracts

  • Fixed an issue where claiming rewards from certain Contracts could cause a duplicate reward notification popup.

  • Fixed an issue where returning Runners, when unlocking Cryo Archive could experience a ‘Contracts Depleted’ error. 

CyberAcme

  • Fixed an issue where the Standard contract “CyAc Guidance: Complex Control” displayed an Enhanced (green) Hardline PR instead of a Deluxe (blue) version in the reward preview.

MIDA

  • Fixed an issue in Expose/Extinguish where the Gantry’s voice could fail to play after completing the terminal hacking step.

Codex

  • Fixed an issue with the “Nightstalker > Spacebucks” entry, only counting singular runs as progress instead of Certs spent.
  • Fixed an issue where the Anomalous Husk entry displayed placeholder text instead of its proper name.
  • Fixed an issue where backing out of the inspect screen for Bungie Rewards items would result in an empty menu.

Sentinel

  • Fixed an issue where “Information Economy” could not be completed after meeting the requirements.
  • Increased the time allowed to defeat a target after an interception for the “Riposte” challenge.

Armory

  • Fixed a tooltip that would advise purchasing ammo and healing items that weren’t present.
  • Fixed an issue where the Traxus Prestige mod “Triple Barrel” showed the wrong cost, requiring 3 Reflex Coils instead of 1 Synapse Cube.
  • Fixed an issue that prevented the Night Marsh Sponsored Kit from appearing in the Armory if under Runner Level 3.
  • Fixed an issue where the NuCaloric Armored Barter was showing the incorrect upgrade icon.
  • Fixed an issue where the Enhanced Duck and Cover Schema did not provide an Armory offer after exfil.

Zones

Dire Marsh (Night)

  • Fixed an issue that prevented some landmarks from showing up on the mini-map.
  • Fixed an issue in Maintenance where certain Cert containers could be opened without spending any Certs.

Stability

  • Fixed several stability issues that could occur when selecting nodes in the faction upgrades screens.

Localization

  • Fixed various spelling and grammar issues for the multi-select tooltips.
  • Fixed an issue where Thief’s voice over lines could duplicate while using the Russian language.

物品经济

  • 小幅降低了尊享护盾在所有途径掉落的概率。
  • 小幅降低了尊享武器模组在所有途径掉落的概率。
  • 修复了在撤离时，方案奖励的信用点与标注数值不匹配的多个问题。

战斗

武器

  • 修复了在将武器模组从一件已装备武器拖拽到另一件时，即使两件武器都支持该模组，仍显示模组不兼容的问题。
  • 修复了D54战斗手枪尊享模组特性“亡命者枪托”中，通过频繁点击可使超级爆发持续到整个弹匣打空的问题。

芯片

  • 修复了余震特性在受到爆炸伤害后过快失效的问题。

用户界面和体验

  • 修复了准备界面中按键通话失效的问题。
  • 更新了物品提示，现在当大厅启用多选时，不再显示部分操作选项。
  • 为夜间沼泽预览新增了缺失的区域状态效果“黑暗降临”。
  • 修复了低温维护室（低温档案室1）钥匙描述中错误标注开锁步骤的问题。
  • 修复了在使用AMD FSR且分辨率缩放设为“质量”或更低时，大厅出现重影/视觉异常的问题。
  • 修复了在疾行中途开启主播模式时，HUD显示名称不一致的问题。
  • 修复了第2赛季高级奖励通票购买界面错误显示为奢享币图标，而非奖励通票兑换券图标的问题。
  • 修复了通过来源链接从“夜幕之声”匕首风格进入奖励通票时，无法选择奖励通票物品的问题。

阵营

关口基因

  • 修复了在解锁所有必要升级后仍无法领取巅峰6的问题。

合约

  • 修复了在领取部分合约奖励时弹出重复奖励通知的问题。

  • 修复了回归玩家解锁低温档案室后，可能出现“合约已耗尽”错误的问题。 

赛博艾克米

  • 修复了标准合约“赛博艾克米指引：综合设施控制区”中的奖励预览错误显示为强化（绿色）铁律精密步枪，而非豪华（蓝色）的问题。

米达

  • 修复了在“曝光/扑灭”中，完成终端骇入步骤后甘特里的语音可能无法播放的问题。

百科

  • 修复了“夜行猎手 > 太空币”条目仅按单次疾行，而未按消耗认证计算进度的问题。
  • 修复了“异常巨壳”条目错误显示为占位文本，而非正式名称的问题。
  • 修复了在查看Bungie奖励物品检视界面后返回时，菜单变为空白的问题。

哨兵

  • 修复了在满足条件后仍无法完成“信息经济”的问题。
  • 延长了“一弹还一弹”挑战中，拦截完成后击败目标的时间限制。

军械库

  • 修复了提示错误建议购买未提供的弹药和治疗物品的问题。
  • 修复了特莱克斯尊享模组“三联枪管”的费用错误显示为3个反射线圈，而非1个突触立方的问题。
  • 修复了疾行者等级低于3时，夜间沼泽赞助包无法在军械库显示的问题。
  • 修复了纽卡洛里护甲易物升级图标显示不正确的问题。
  • 修复了在撤离后，强化卧倒掩护方案未提供军械库物品的问题。

区域

恐怖沼泽（夜间）

  • 修复了部分地标无法在小地图上显示的问题。
  • 修复了维护部中部分认证容器无需消耗认证即可开启的问题。

稳定性

  • 修复了在阵营升级界面选择节点时可能出现的几个稳定性问题。

本地化

  • 修复了多选提示中的多处错别字和语法错误。
  • 修复了设置为俄语时，盗贼的语音台词可能出现重复的问题。
