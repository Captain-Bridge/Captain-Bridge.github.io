Marathon Update 1.1.0.4

The one about Cryo lasers.

Marathon Dev Team July 7, 2026
updates196 words1 min read

General

  • Sponsored Mode: Zone updated to Dire Marsh (Night).
  • The C.A.R.R.I. Armory has been updated with the following changes.
    • Wares Added:
      • Deluxe Key Template
      • Blue Backpack
      • Self Revive
    • Wares Changed:
      • Lockbox Key Commendations Cost: 10 → 7
      • Reputation Pack Weekly Stock: 1 → 2
      • Rotating Grey Weapon Commendations Cost: 10 → 5
    • Updated the Dire Marsh (Night) Sponsored activity description to clarify the Sponsored Kit loadout requirement.
    • Fixed an issue where players could be matchmade into Trios when the UI displayed Solos.
    • Fixed an issue that could prevent some mailbox messages from being delivered.

Combat

  • Fixed an issue where the Sniper boss targeted audio could play for nearby Runners who weren’t targeted.

Runners

Triage

  • Fixed an issue where the Astrophage style’s face would brightly glow when in the Dire Marsh (Night) Zone.

Zones

Dire Marsh (Day)

  • Fixed an issue where Lockdown activities were failing to spawn in some regions.

Cryo Archive

  • Fixed an issue where Runner models could still hit tripwires (green lasers) when sliding under or jumping over them.

Stability

  • Fixed an issue where a small number of players could get stuck in a Weasel disconnect loop that would prevent the ability to consistently login.

综合

  • 赞助模式：区域已更新为恐怖沼泽（夜间）。
  • C.A.R.R.I.军械库已更新，具体更改如下。
    • 新增商品：
      • 豪华钥匙模板
      • 豪华背包
      • 自我复活
    • 变更商品：
      • 锁箱钥匙嘉奖消耗：10 → 7
      • 声望包每周库存：1 → 2
      • 轮换灰色武器嘉奖消耗：10 → 5
    • 更新了恐怖沼泽（夜间）赞助活动描述，以明确赞助包配装要求。
    • 修复了UI界面显示单人模式时，玩家可能会被匹配到三人组的问题。
    • 修复了部分邮件无法送达的问题。

战斗

  • 修复了狙击首领的目标音效可能会对附近非目标疾行者播放的问题。

疾行者

救援

  • 修复了在恐怖沼泽（夜间）区域时，噬星体风格的面部会发光的问题。

区域

恐怖沼泽（白天）

  • 修复了部分区域封锁活动无法生成的问题。

低温档案室

  • 修复了疾行者模型在滑行或跃过绊索（绿色激光）时仍会碰到绊索的问题。
