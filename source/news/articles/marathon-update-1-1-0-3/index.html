Marathon Update 1.1.0.3

The one about Cryo Archive Sponsored Kits.

Marathon Dev Team June 23, 2026
updates211 words1 min read

Item Economy

  • Fixed an issue where barter options were not populating for the Compiler Ganglion.
  • Swapped Arachne and Sekiguchi VIP upgrade costs so Synapse Cubes can be used for certain upgrades instead of the previous prestige salvage.
  • Adjusted MIDA’s final Capstone to now grant ANTI_VIRUS.EXE, providing 2 minutes of Anti-Virus protection at the beginning of a run. Previously was providing Cardio Kick that did not stack with the Dexterity track perk.

Combat

Weapons

KKV-9SD

  • Fixed an issue where the KKV-9SD would cease firing while holding down the trigger on certain controllers.

User Interface and Experience

  • Fixed an issue where stickers were not correctly showing up on the Retro_Remix WSTR Shotgun style.

Zones

Perimeter

  • Fixed an issue where Solo queue was selectable on Ranked.
    • Developer Note: The intention for players queueing into Ranked is the same as Season 1, which is that crews of all sizes enter the same match. It was not intended that players could select Solo queue in Ranked, and this did not function as players expected.

Dire Marsh (Night)

  • Fixed an issue that was preventing Reverbs from spawning consistently on standard Night Marsh.

Cryo Archive

  • Fixed an issue where the Cryo Archive sponsored kit could not be taken into Cryo Archive.

Stability

  • Fixed a few server stability issues.

物品经济

  • 修复了编译者神经节的易物选项无法显示的问题。
  • 替换了阿拉克尼和关口VIP升级所需的资源，现可使用突触立方进行某些升级，而无需像以前使用尊享回收资源。
  • 调整了米达的最终巅峰，现可提供杀_毒.EXE，在疾行开始时提供2分钟的杀毒保护。之前提供的心肺复苏无法与灵巧特性叠加。

战斗

武器

KKV-9SD

  • 修复了在某些控制器上按住扳机时KKV-9SD会停止射击的问题。

用户界面和体验

  • 修复了复古混音WSTR霰弹枪风格的贴纸显示不正确的问题。

区域

外围区域

  • 修复了排名模式中可以选择单排的问题。
    • 开发者提示：玩家进入排名模式的初衷与第1赛季相同，即所有规模的团队都将进入同样的比赛。让玩家在排名模式中选择单排并非我们本意，而这一设定也未能达到玩家预期。

恐怖沼泽（夜间）

  • 修复了标准夜间沼泽无法稳定生成余响的问题。

低温档案室

  • 修复了低温档案室赞助包无法带入低温档案室的问题。

稳定性

  • 修复了一些服务器稳定性问题。
