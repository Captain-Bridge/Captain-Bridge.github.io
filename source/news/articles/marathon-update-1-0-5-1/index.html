Marathon Update 1.0.5.1

The one about Cryo Archive (Cont.)

Marathon Dev Team March 24, 2026
updates182 words1 min read

Item Economy

  • Unowned Cryo Archive shell styles are now visible within the Customization menu to mirror Codex visibility.

Runner Shells

Thief

  • Fixed an issue that allowed the Pickpocket Drone’s cooldown to be refreshed by pressing the Prime Ability button as it times out.

Zones

Outpost

  • Fixed out of bounds geometry that would allow unintended entry into the Pinwheel.
  • Fixed geometry that would allow out of bounds access near Dormitories.

Cryo Archive

  • Fixed some environmental security door wall art render distance.
  • Fixed some geometry that would allow loose items to fall through the environment.

User Interface and Experience

Pings

  • Fixed an issue causing exfils on overhead map to disappear when pinging and unpinging repeatedly.

Codex

  • Fixed and updated localization for Ranked cosmetic challenges for Gold and Diamond.

Contracts

  • Fixed an issue causing the contract reroll refresh timer to show as +1 week.

CyberAcme

  • Fixed an issue that was blocking contracts and showing as “Contract Disabled.”

Stability

  • Fixed a variety of issues that were causing crashing, kicked to lobby errors, and general stability.

Localization

  • Fixed various errors in localization for Cryo Archive Codex entries.

物品经济

  • 现在，未拥有的低温档案室外壳风格会在自定义菜单中显示，使其与百科中的显示一致。

疾行者外壳

盗贼

  • 修复了窃贼无人机冷却时间即将结束时，按下主要技能按钮即可刷新冷却时间的问题。

区域

前哨

  • 修复了超出边界的几何体会导致玩家意外进入风轮的问题。
  • 修复了宿舍附近超出边界几何体的访问权限问题。

低温档案室

  • 修复了部分环境安全门墙饰的渲染距离。
  • 修复了部分几何体可能导致散落物品穿过环境的问题。

用户界面和体验

标记

  • 修复了在俯视地图上反复标记和取消标记会导致撤离点消失的问题。

百科

  • 修复并更新了黄金和钻石排名装饰挑战的本地化内容。

合约

  • 修复了合约刷新计时器显示+1周的问题。

赛博艾克米

  • 修复了合约被阻止并显示“合约已禁用”的问题。

稳定性

  • 修复了导致游戏崩溃、被踢回大厅错误以及整体稳定性的各种问题。

本地化

  • 修复了低温档案室百科条目的本地化错误。
